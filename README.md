## 최악의 플레이를 박제해 보세요!

친구들과 시간을 보내다가 자신의 치부가 드러난적 있으신가요?

```
🤦‍♂️ 한 번의 실수를 매번 놀리는 친구들

🤦‍♀️ 자신들은 완전무결한 줄 아는 친구들

🤔 나를 🐃취급하는 친구들
```


걱정 마세요! 온갖 음해에 시달린 당신에게 **SeolCat**은 좋은 공격 수단을 제공합니다.

```

👨‍👩‍👦 미래 후손에게 가보로 남기고 싶은 플레이
```
